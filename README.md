SharkShell is a secure, modern, web-based SSH terminal and keystore manager designed for infrastructure access management.

## Features
- **Secure Web Terminal:** Connect to remote servers directly from your browser using robust Socket.IO streams.
- **Encrypted Keystore:** Store and generate SSH keys using AES-256-GCM encryption at rest. 
- **Host Management:** Keep track of all your server connections and defaults.
- **Glassmorphic UI:** A beautifully designed interface built with React and modern CSS.

## Architecture
This project is built using a modern decoupled architecture:
- **Backend:** NestJS (Node.js) / PostgreSQL
- **Frontend:** React / Vite
