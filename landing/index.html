Features

Everything you need to manage
remote servers

Enterprise-grade tools, designed for simplicity.

Multi-Tab Terminals

Open multiple SSH sessions in tabbed terminals. Switch between servers instantly without losing your work.

Workspaces

Organize your sessions into named workspaces. Sessions persist through page refreshes — click to reconnect.

Encrypted Keystore

AES-256-GCM encryption for all stored credentials. Private keys and passwords are encrypted at rest.

SSH Key Management

Generate, upload, download, and organize SSH keys. Support for ED25519, ECDSA, and RSA key types.

Host & Key Groups

Color-coded groups with glowing visual indicators. Organize servers by environment and connect entire groups at once.

Session Persistence

Sessions survive page refresh. Your terminals stay alive — just click to reconnect and pick up where you left off.

Security

Enterprise-grade
security built in

Your credentials are protected by industry-standard encryption.

AES-256-GCM

All passwords and private keys are encrypted at rest using AES-256-GCM — the same standard used by governments and banks.

RSA JWT Auth

Auto-generated 2048-bit RSA keypair for token signing. Secure session management with RS256 algorithm.

First-Time Lockdown

Seamless admin setup on fresh installs. Open registration is automatically blocked after the first admin account is created.

Non-Root Container

Docker image runs as an unprivileged user. Secrets are stored in persistent volumes and survive container restarts.

Get Started

Deploy in
60 seconds

One command. That's it.

1

Recommended: Docker Compose

Download the compose file, run the init script, and start.

terminal 
# Download
curl -sO https://raw.githubusercontent.com/sushilkumarsahani41/SharkShell/main/docker-compose.prod.yml
curl -sO https://raw.githubusercontent.com/sushilkumarsahani41/SharkShell/main/init.sh
chmod +x init.sh

# Setup & Run
./init.sh
docker compose -f docker-compose.prod.yml up -d
2

Or: Docker Run (Bring Your Own Database)

If you have an external PostgreSQL database, run the standalone container.

terminal 
docker run -d -p 8080:80 \
  -e DB_HOST=your-db-host \
  -e DB_USER=your-user \
  -e DB_PASSWORD=your-password \
  -e DB_NAME=sharkshell \
  -v sharkshell-secrets:/app/secrets \
  greatsharktech/sharkshell:latest
3

Open & Create Admin

Navigate to http://localhost:8080. On first visit, you'll see the admin setup screen — create your account and you're ready to go. 🚀
